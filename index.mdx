## Why Datalinks?

Datalinks eliminates the complexity of integrating external data sources with Alfresco. Define your database connections, configure your data mappings, and let users link document properties to live database records—all through an intuitive interface.

## How It Works

<Steps>
  <Step title="Define Your Datalink">
    Create a JSON configuration file specifying your database endpoint, authentication, and column mappings
  </Step>
  <Step title="Deploy to Alfresco">
    Place the configuration in the datalink folder and start Alfresco—datalinks are registered automatically
  </Step>
  <Step title="Link Document Properties">
    Users can right-click documents in Alfresco Content App and select "Edit Datalink" to link properties to database records
  </Step>
</Steps>
